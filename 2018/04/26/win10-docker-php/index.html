Win10 搭建基于 Docker 的 PHP 开发环境

发表于 2018-04-26 | 分类于 教程

PHP 开发环境最佳实践

必备条件

  • Win10 Pro

安装

下载

  • Docker For Windows
  • Docker For Windows

开启 Win10 的 Hyper-V

  • BIOS 中开启虚拟化支持
  • 控制面板->程序->启用或关闭 Windows 功能->选中 Hyper-V
  • 安装 Docker For Windows
    • 确认开启 Hyper-V，否则启动 Docker 的时候会提示开启 Hyper-V
    • 在 Docker 的 Settings-> Advanced 调节 VM 内存大小，否则启动时可能会提示内存不足导致启动失败
    • 在 Docker 的 Setting->Share Drivers，勾选你需要的盘，否则可能会遇到 Driver has not been shared 的问题
    • 在 Docker 的 Settings->Deamon 中设置国内镜像地址 https://registry.docker-cn.com

注意

  1. 如果在容器中起用了 MySQL 服务，则项目中 db 配置的 dsn 的 host 应该为 mysql
  2. 导入 sql 文件到 mysql 中的命令为 docker exec -i [CONTRAINER_NAME] bash -c "mysql -uroot -proot avatar" < open_initial.sql
    MySQL 服务的 CONTRAINER_NAME 可进入 laradock 目录下，执行 docker ps 命令进行查看。
